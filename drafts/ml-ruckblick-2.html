Deep Neural Networks are Easily Fooled

Der Input für Neuronale Netze zur Bild-Klassifikation kann auf zwei Arten manipuliert werden:

  1. Ein Bild, auf welchem ein Mensch nichts erkennt (z.B. white noise) bekommt einen hohen Score für ein Objekt
  2. Ein Bild auf welchem ein Mensch eine Sache eindeutig erkennt (z.B. "Katze") wird so manipuliert, dass das NN mit einem hohen score etwas anderes ausgibt (z.B. "Schulbus").
